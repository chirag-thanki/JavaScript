Variables can be defined with different names and they have certain rules while defining.

  1. They should not start with numbers
  2. They can start with a dollar ($) sign and underscores (_)
  • E.g. myName = "Chirag";
  • E.g. $name = "Chirag";
  • E.g. $$name = "Chirag";
  • E.g. _name = "Chirag";
  • E.g. __name = "Chirag";
  • E.g. _Name = "Chirag";
